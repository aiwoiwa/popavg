# ポップ君のサイズの計算式
## textage
単位: ピクセル
70.gif: 134 * 32
s.gif: 36 * 4
w.gif: 13 * 4
1小節に70.gifを4つ配置しているため，70.gifのサイズは
横: 134 = 13 * 7 + 36 + 7(8ノーツの間に1px分を確保している)
縦: 32 = 4 * 8
である．
